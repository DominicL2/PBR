# PBR Test Application
------
This application is draw to 3D graphics model using PBR

## Supported PBR Model
* Diffuse
  * Lambertian 
  * Oren-Nayar (when choose Cook-Torrance)

* Specular
  * Phong
  * Blinn-Phong
  * Cook-Torrance

## Supported Graphics File
* Obj (only Phong and Blinn-Phong)
* FBX

## 3rd party list
* ASSIMP
* Qt 5.6 (uisng QML)
* OpenGLES 2.0
